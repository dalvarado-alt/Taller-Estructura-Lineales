## Descripción del Taller
Este repositorio contiene la solución al taller de **estructuras lineales**, desarrollado en Java y ejecutado desde consola.  
Se implementan ejercicios con:
- Arreglos unidimensionales
- Arreglos bidimensionales (matrices)
- ArrayList

El objetivo del taller es practicar conceptos fundamentales de programación como:

* Manejo de arreglos y matrices
* Uso de `ArrayList`
* Lectura de archivos
* Uso de ciclos (`for`, `while`)
* Manipulación de datos
* Programación orientada a objetos

Los ejercicios se encuentran **organizados en carpetas según el tema**.
Por ejemplo:

* `arraylist` → ejercicios que trabajan con `ArrayList`
* `matrices` → ejercicios que trabajan con matrices
* `arreglos` → ejercicios que utilizan arreglos

## Instrucciones para ejecutar el programa

1. Clonar el repositorio en su computador.

2. Abrir el proyecto en un entorno de desarrollo como **Visual Studio Code** o **NetBeans**.

3. Ubicarse en la carpeta del ejercicio que desea ejecutar.

4. Compilar el archivo `.java`.

5. Ejecutar la clase correspondiente.

Ejemplo desde la terminal:

```
javac Ejercicio1.java
java Ejercicio1
```

Cada ejercicio puede ejecutarse de forma independiente.
